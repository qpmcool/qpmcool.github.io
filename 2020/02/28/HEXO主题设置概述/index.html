HEXO主题设置概述

大秦

HEXO

发布于：2020年2月28日

更新于：2020年2月29日

  • HEXO默认设置
  • HEXO主题设置
  • HEXO页面配置

HEXO默认设置

只提及常用设置，其他设置可以参考HEXO网站
打开主目录下的_config.yml文件进行网站配置

HEXO主题设置

打开themes\主题目录\_config.yml进行主题配置
一般开发者都对文件进行了非常好的标注，遵循标注的内容，即可进行很优秀的配置。资源库说明见HEXO入坑指南
打开themes\主题目录\languages\**.yml进行语言和部分显示信息的配置（例如版权，协议等），部分需要配合css文件进行修改，可忽略。

封面设置

封面菜单为首页主菜单，必要时，可以做成多级的。

导航栏设置

菜单链接，需要对应的文件支持，如果source文件夹下没有对应的文件，需要新建。
终端 hexo new page 对应目录链接
完成后，可以进行装饰，支持html5的标签。

HEXO页面配置

Blog页面各种链接可能无法使用，需要进行配置生成。
独立页面都需要手动创建md文件或用上面介绍的命令生成。
通常布局模板有

取值 含义
page 独立页面
post 文章页面
category 分类页面
tag 标签页面
links 友链页面
list 列表页面

关于页面

手动创建文件：source/about/index.md

分类页面

手动创建文件：source/categories/index.md

标签页面

手动创建文件：source/tags/index.md

友链页面

手动创建文件：source/friends/index.md

404页面

手动创建文件：source/404.md
